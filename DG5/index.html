Jump The Gun

  • Originality: The game is a platformer that aims to use the recoil from the player's weapon as a primary method of movement.
  • Technical Merit: The game uses a tilemap to build the level and directly translates the tiles there to tangible objects.
  • Prototype Postmortem: My original plan was to use P2 physics so that I could shoot a projectile at a platform and then connect the player (via their weapon) and the platform where it was shot with a spring with no springiness. This was completely scrapped when I found out that phaser weapons would never use anything but Arcade physics. These different physics systems would never interact so any bullet/platform interaction is impossible. I also considered using box2d that looked like it had weapon/rope/platform physics that i wanted to implement, but came at a price over $0, too much for me. Having never gotten past the weapon system, the rest of the game (more level/enemies/score) has not yet been implemented. Additionally, Recoil was also bugged, always accelerating the player in the (same) wrong direction or not doing anything.
